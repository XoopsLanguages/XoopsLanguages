Bantuan: Mesej Peribadi

Huraian

Module mesej peribadi adalah untuk menghantar mesej peribadi diantara pengguna

Proses Install/uninstall

Ikut arahan biasa proses installation - extract /pm folder ke dalam direktori ../modules. Install module tersebut menerusi Admin -> System Module -> Modules.

Arahan lanjut untuk proses install module boleh didapati dalam Manual Operasi XOOPS

Arahan Operasi

Untuk set up module ini anda perlu:

  • Anda masuk dalam Kotak Masuk anda dari muka depan atau dengan "Pergi kepada module" dalam bahagian pentadbiran mesej peribadi.
  • Konfigurasi tetapan anda untuk module ini (Lihat ‘Preferences’)
  • Anda boleh buat proses mass deleting mesej anda dengan menggunakan fungsi Hapus Mesej (Prune Messages).
