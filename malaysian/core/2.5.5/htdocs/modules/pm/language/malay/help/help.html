Help: Private Messaging

Description

The PM module is for private messaging between users

Install/uninstall

No special measures necessary, follow the standard installation process – extract the /pm folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Operating instructions

To set up this module you need to:

i) You enter your Inbox directly from the front page, or by clicking on the top link "Go to module" in the PM's Admin area

ii) Configure your preferences for the module (see ‘Preferences’) and optionally the PM block if you intend to use it (see ‘Blocks’)

iii) You can do mass deleting of your messages by using the "Prune Messages" tab
