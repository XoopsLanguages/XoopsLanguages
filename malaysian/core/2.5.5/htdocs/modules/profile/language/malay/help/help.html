Bantuan: Profile

Huraian

Module untuk menguruskan medan tambahan untuk profil pengguna.

Proses Install/uninstall

Ikut arahan biasa proses installation - extract /profile folder ke dalam direktori ../modules. Install module tersebut menerusi Admin -> System Module -> Modules.

Arahan lanjut untuk proses install module boleh didapati dalam Manual Operasi XOOPS

Arahan/Operasi

To set up this module you need to:

  • Tetapkan pilihan anda untuk module ini (Lihat Tetapan atau ‘Preferences’)
  • Edit Kategori sedia ada atau tambah kategori baru.
  • Edit medan sedia ada atau tambah yang baru. Disini anda boleh nyatakan medan mana yang boleh dilihat dalam kategori dan samada ia akan boleh dilihat dalam proses pendaftaran.
  • Nyatakan susunan Langkah Pendaftaran.
  • Set permissions untuk medan - yang mana yang boleh diedit, mana yang boleh dicari dan sebagainya.
