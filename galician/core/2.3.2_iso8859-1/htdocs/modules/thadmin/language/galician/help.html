Resumen

  • O módulo posibilita o uso de themes para o área de administración.

Instalar

  • Para instalar o módulo, siga estos 3 pasos :
    • Suba ThAdmin ó directorio dos módulos e instale o módulo.
    • En Configuración Xeral, elixa 'thamin' para os parámetros do CPanel GUI

Engadir novo deseño

  • Copie/pegue e renomee o deseño por defecto
  • Modifique o seu deseño e actualice o seu módulo. O deseño aparecerá dentro das configuracións do módulo
