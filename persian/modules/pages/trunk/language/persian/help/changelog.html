Correction du bug suivant : lorsque le module pages est défini comme module par défaut pour la page d'accueil, si vous définissez un affichage de blocs uniquement pour cette page d'accueil, ces blocs s'afficheront pour toutes les pages du module Pages.

Les nouvelles fonctionnalités intégrées dans cette version pourront rendre plus délicate le clonage de ce module, une prochaine version s'attachera à corriger cela avec l'éventualité de l'intégration d'un script de clonage.
