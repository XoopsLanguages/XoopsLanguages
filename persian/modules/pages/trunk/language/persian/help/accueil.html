This module which could be describe as a module for building free Content or as a multi-fonction pocket knife. This is a module for publish independent pages (or not).

Free content

If you want to build a site to be present on Internet and to present your activity (trade, liberal profession, etc...) this module associated with a contact module and a map module (for referencing) is completely available to fill this task.

multi-fonction pocket knife

It allows to put in your web site :

  • a full page about privacy rules on your site for the data of your members or visitors
  • the rules of your site
  • a presentation of the activities of your company. When you buid a website it's often the main module which the site is build.

Advantages

  • the data published in theses pages are accessible by the search function of XOOPS.
  • easy to use : the module supports the HTML language but most of the WYSIWYG installed in XOOPS are available. You write and publish, that's all !
  • all pages can be added as a sub-link in the main menu block
  • allows to publish all that does not go in the other XOOPS modules
  • include the hack for the title of the pages so that each page has a specific title
  • include the metagen engine : the Metagen function improve referencing of the pages including the tag Meta-Keywords with the words included in the pages
  • support the RSS standard for your pages and/or possibility of including them on another web site with a small Javascript script
