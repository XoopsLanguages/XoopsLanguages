Menu administration : Page Index

Cette page représente l'affichage par défaut lorsque vous arrivez dans la partie administration du module, elle vous affiche différentes informations comme le nombre de pages déjà créées, l'accès à la configuration des pages spéciales (informer un ami, faire un lien sur le site).

Dans la partie inférieure de la feuille, un tableau présente les pages déjà créées avec les informations essentielles sur chacune d'elles, avec la possibilité de les éditer directement ou d'effectuer une suppression.
