Hjælp: Vedligeholdelse

Beskrivele

Vedligeholdelse indeholder flere grundlæggende funktioner til at hjælpe dig med at vedligeholde din XOOPS installation og holde det up-to-date:

- ryd cache mappen
- tøm session table
- udfør vedligeholdelse af databasen (analysér, kontrollér, optimer, and reperer)

Du kan også udføre en "dump" af hele databasen eller enkelte tabeller til backup formål.
