Installation de XOOPS

Première installation

Préface :

XOOPS est un systême de gestion de contenu orienté-objet open source écrit en PHP. C'est un outil idéal pour développer des sites Web de petite à grande communauté dynamique, portails d'entreprise en intranet, portails entreprises, weblogs et plus encore.

XOOPS est distribué sous les termes de la GNU License Générale Publique (GPL) et est libre d'être utilisé et modifié. Il est libre d'être redistribué tant que vous respectez les modalités de redistribution de la GPL.

Exigences

  • WWW Server (Apache, IIS, Roxen, etc)
  • PHP 5.2.0 ou supérieur
  • MySQL 5.1 ou supérieur

Avant l'installation

  1. Configurez correctement le WWW du serveur, PHP et la base de données.
  2. Préparez une base de données pour votre site XOOPS.
  3. Préparez votre Compte d'utilisateur et donnez-lui l'accès à la base de données.
  4. Ouvrez en écriture le dossier /uploads/ et le fichier mainfile.php (CHMOD 777)
  5. Pour des raisons de sécurité, nous vous conseillons fortement de déplacer les deux dossiers /xoops_lib/ (pour les bibliothèques XOOPS) et /xoops_data/ (pour les données XOOPS) en dehors de la racine et de changer leurs noms.
  6. Ouvrez en écriture /xoops_data/ ; Créez (si ils n'existent pas) les dossiers /xoops_data/configs/, /xoops_data/caches/, /xoops_data/caches/xoops_cache/, /xoops_data/caches/smarty_cache/ and /xoops_data/caches/smarty_compile/ et ouvrez-les aussi en écriture.
  7. Activez les cookies et le JavaScript de votre navigateur.

Installation locale

Si vous exécutez un environnement de développement ou de test, assurez-vous que vous avez suivi les exigences précédentes. Une fois que cela est fait, copiez le contenu du répertoire htdocs (à partir du fichier de distribution de XOOPS ou du SVN) vers le chemin du document racine de votre environnement web (après le www/). Une fois que les fichiers sont copiés, vous pouvez démarrer l'installation en tapant « http://votresite.com » (pas littéralement, mais le vrai chemin de votre site). Cela démarrera le processus d'installation.

Installation sur une plateforme hébergée

Si vous exécutez dans un environnement hébergé, décompressez les fichiers XOOPS localement ou sur le serveur si vous avez telnet ou l'accès SSH. Une fois que c'est fait, assurez-vous de déplacer ou de copier tous les fichiers XOOPS dans le répertoire htdocs à la racine (votre fournisseur vous fournit habituellement cet endroit avec les directives). Une fois que les fichiers sont copiés, vous pouvez démarrer l'installation en tapant « http://votresite.com ». Cela démarrera le processus d'installation.
