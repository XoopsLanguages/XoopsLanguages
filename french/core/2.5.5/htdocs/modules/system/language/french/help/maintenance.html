Aide : Maintenance

Description

La partie "Maintenance" fournit plusieurs fonctions simples pour maintenir sain votre système XOOPS :

  • Suppression des fichiers temporaires dans les dossiers de cache
  • Suppression des enregistrements dans la table des sessions
  • Réalisation des opérations d'analyse, de vérification, d'optimisation et de réparation des tables de votre base de données

Vous pouvez également effectuer une sauvegarde de toute votre base de données, de certaines tables uniquement, ou de celles d'un module particulier.
