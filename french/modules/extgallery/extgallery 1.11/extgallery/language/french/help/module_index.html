Aide : eXtGallery

Description

eXtGallery est un module de galerie photo internet puissant pour XOOPS

Categorie et Album


Modifier ou supprimer des catégories publiques :
Vous pouvez trouver une liste de toutes les catégories et albums disponibles sur votre site internet. Le nombre à la fin du nom de la catégorie est le poids de la catégorie.

Ajouter une catégorie de publique :
Vous pouvez uniquement ajouter une catégorie dans une catégorie sans photo. Seule une catégorie valide est affichée dans la liste.

Photos

Ajouter des photos par lot :
Chemin de dossier du lot : /opt/lampp/htdocs/local/test/tdm1/modules/extgallery/batch/
En cliquant sur le bouton Valider, toutes les photos dans le dossier de lot seront téléchargées.

Reconstruire les vignettes :
Sélectionnez la catégorie où la vignette devra être reconstruite. Cette fonctionnalité est utile si vous souhaitez modifier la taille ou la qualité de la vignette

Modifier ou supprimer une photo:
Sélectionnez la catégorie dans laquelle vous souhaitez modifier ou supprimer des photos.

Permissions

Choisir les permissions :
Veuillez vérifier et mettre à jour toutes les permissions. Si vous souhaitez que les utilisateurs voient vos albums et vos photos, vous devez sélectionner l'autorisation pour tous les groupes

Masque d'autorisations publiques :
Ici vous pouvez définir la valeur par défaut des autorisations qui seront appliquées à une nouvelle catégorie. Ne donnez pas plus d'autorisations ici parce que toutes les nouvelles catégories en hériteront.

Autorisation d'accès :
Vous permet de restreindre l'accès à cette catégorie. Un utilisateur restreint ne peut pas voir une catégorie sauf s'il dispose des autorisations pour le faire.

Autorisation de vote :
Permet de limiter la fonctionnalité des votes. Un utilisateur restreint ne peut pas évaluer dans une catégorie mais peut voir la cote d'une image.

Autorisation publique des carte électroniques :
Vous permet de restreindre la fonctionnalité d'envoi des cartes électroniques. Les adresses IP de tous les expéditeurs de cartes électroniques sont consignées.

Autorisation de téléchargement publique :
Vous permet de restreindre le téléchargement d'une image de grande taille. Si un utilisateur est autorisé à télécharger, ils aura également accès aux téléchargements de grandes images si « Sauvegarder les grandes photos » est activé dans les préférences.

Autorisation de téléchargement des originaux :
Permet de limiter le téléchargement des images originales. Si un utilisateur est autorisé à télécharger un original, ils peut accéder à l'image d'origine (sans bordure et filigrane et non redimensionnée) si « Sauvegarder les grandes photos » est activé dans les préférences.

Autorisation d'envoi publique :
Permet de limiter le téléchargement et la soumission de nouvelles images pour les albums publiques.

Autorisation des approbations automatiques publiques :
Vous permet d'empêcher l'affichage automatique de l'image publique soumise jusqu'à ce qu'elle soit approuvée par un administrateur. Sélectionnez la catégorie où les téléchargements des utilisateurs publics ne doivent pas être approuvés. Les utilisateurs publics doivent être autorisés à soumettre des photos pour permettre à cette autorisation.

Autorisation d'affichage de l'album :
Vous permet de ne pas afficher les noms des catégories, mais d'avoir toujours accès à l'image. Vous pouvez par exemple utiliser des photos dans ces catégories, pour le contenu de votre site.

Configuration de l'album :
La sélection d'effets va changer l'affichage des photos lorsque vous cliquez sur les vignettes. L'effet Ajax ou Normal peut être sélectionné.

Normal : Pas d'effets
Visionneuse : Utilise l'effet visionneuse
Superposition : utilise l'effet de superposition
Info-bulle : utilise l'effet de l'info-bulle
Boîte fantaisie : Utilise l'effet boîte fantaisie
Jolie Photo : Utilise l'effet Jolie Photo
