Schéma

  • Ce module vous donne la possibilité de gérer du contenu pour le CMS XOOPS.
  • Il permet de lier automatiquement votre contenu à un menu.

Accueil

  • Visionner l'ensemble des pages présentes dans le module.
  •    : Réordonner les pages et les sous-pages.
  •  Visualiser,  éditer et  supprimer une page.

Contenu

  • Ajouter du contenu au module content (Pages, Liens, Sections ou Séparateurs de texte).
  • Un lien dynamique peut être affiché dans le bloc de navigation pour chaque type de contenu.
  • "Contenu": création de pages.
    Vous devez définir un titre de page, un titre de menu et saisir le texte.
    Vous pouvez aussi choisir l'alias qui sera utilisé lors de l'affichage du titre dans les url.
  • "Lien": ajouter un lien interne ou externe au menu dynamique.
    Par exemple, vous pouvez simplement créer un lien vers la page d'accueil de votre site.
  • "Section": ajouter une entrée non cliquable dans votre menu afin de l'organiser en sections.
  • "Séparateur": création d'un séparateur dans le menu dynamique.
