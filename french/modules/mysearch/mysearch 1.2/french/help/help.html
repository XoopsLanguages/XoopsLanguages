Help: MySearch

Description

Avec ce module, vous pouvez apprendre ce que cherchent les personnes qui consultent votre site Web.

Installation/désinstallation

Aucune mesure spéciale n'est nécessaire, suivez la procédure d'installation standard - envoyez le dossier /mysearch dans ../modules . Installez le module via Admin -> Système -> Modules.

Des instructions détaillées sur l'installation du module sont disponibles dans le Manuel des opérations de XOOPS
