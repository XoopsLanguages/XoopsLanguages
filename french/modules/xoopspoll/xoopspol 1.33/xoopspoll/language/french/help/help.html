Aide : XOOPS Poll

Description


Le module de sondages peut être utilisé pour afficher des formulaires de sondage interactifs sur votre site. Chaque sondage peut afficher une question sur laquelle les visiteurs peuvent « voter ». Cela peut être un précieux moyen de recueillir les commentaires de votre communauté. Les sondages peuvent être configurés pour permettre un seul ou plusieurs choix et les visiteurs peuvent voir les « résultats » à jour.

Installation / désinstallation


Aucune mesure particulière n'est nécessaire, suivez le processus d'installation standard : envoyez le dossier xoopspolls dans le répertoire .../modules. Installez le module par Admin-> Système -> Modules.

Des instructions détaillées sur l'installation de modules sont disponibles dans le Manuel des opérations de XOOPS

Instructions d'exploitation


Le module de sondages est très simple à configurer et à utiliser. Vous avez éssentiellement besoin de :

i) Créer un ou plusieurs sondages pour que les gens votent (Administration de sondages -> Ajouter un sondage)

ii) Afficher le bloc de sondages quelque part sur votre site Web (ce n'est pas strictement nécessaire – on peut aussi y accéder grâce au lien « Sondages » dans le menu principal – mais afficher le bloc de sondages dans un endroit bien en vue peut encourager les gens à voter)

iii) S'assurer que les groupes d'utilisateurs concernés ont des droits d'accès au a) module de sondages et b) bloc des sondages afin qu'ils puissent les voir et voter. Les instructions détaillées sur la configuration des droits d'accès pour les groupes d'utilisateurs sont disponibles dans le Manuel des opérations de XOOPS

iv) La plupart des fonctions de gestion de sondages existants se trouvent dans l'administration des sondages -> Liste des sondages.
