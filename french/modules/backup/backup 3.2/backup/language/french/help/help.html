Aide : Backup

Description

Un module qui vous permet de facilement sauvegarder et / ou restaurer votre base de données MySQL.

Caractéristiques :
=============
1. Sauvegarde les données Xoops et les stocke de trois façons : sur le serveur, téléchargement direct (administrateur uniquement) ou envoie par mail (URL ou pièces jointes).
2. Les tables peuvent être sélectionnées pour être exportées séparément. Utile pour de grandes tables.
3. Les dossiers stockés sur le serveur local peuvent être configurés.
4. La sauvegarde peut être faite par administrateur ou par Cronjob.
5. Restaure les données à partir de fichiers de base de données XOOPS exportés, à partir du module ou par tout autre script de gestion de base de données comme phpmyadmin.
6. Capable d'importer de gros fichiers par sessions multiples.
7. La restauration peut être effectuée avec date de restauration, par des non-webmestres, utile pour les utilisateurs de Xoops.
8. Les problèmes de délais d'attente ont été efficacement résolus.

Installation / désinstallation


Créer un dossier (upload / backup par exemple) pour stocker les fichiers de sauvegarde.

Mettre le dossier en chmod 777.

Installer le module comme d'habitude dans xoops (admin système - modules).

Si vous souhaitez créer une tâche Cron pour que la base de données puisse être automatiquement sauvegardée, l'URL doit être : UrlDeVotreXoops/modules/backup/admin/backup.php

Opérations
