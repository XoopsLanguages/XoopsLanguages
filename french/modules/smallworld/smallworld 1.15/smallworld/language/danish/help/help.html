Hjælp: SmallWorld

Beskrivelse

Smallworld er et socialt netværk system. Inspireret af de Srinivas Tamada tutorials om Facebook scripting. Basissystemet er blevet omskrevet, tilpasset, og stærkt udvidet til XOOPS af Culex Modulet byder på kan lide/kan ikke lide, bookmarking, væg system, billedgalleri, og benytter XIM hvis det er installeret. På adminsiden kan man slette brugere, udelukkelse af bestemte brugere baseret på tid. En nedtælling vises på brugerens side og viser tiden der er tilbage .. Og mange flere funktioner. Det er fuldt understøttet af XOOPS version, 2.5.0 og højere.
Smallworld er frigivet under betingelserne i GNU General Public License (GPL), og er gratis at bruge og modificere. Det er gratis at omfordele, så længe du overholder distributionsbetingelserne i GPL.

Installer/afinstaller


Ingen specielle forudsætninger nødvendige, følg standard installations process - udpak /smallworld til ../modules dir. Installer modulet via Admin -> System Module -> Modules.

Undersøg altid om du har den nyeste udgave af smallworld. Dette kan ses gennem smallworld/admins første tab Tillad cookie og JavaScript i din browser.

Installation på hosted server platform
  • Upload den udpakkede mappe Smallworld til dit module dir.
  • Installer som hvilket som helst andet modul.
  • Gør modulet tilgængeligt for webmaster, registrede brugere eller andre (admin/moderators etc etc).
Opgradering fra tidliger version
  • Undersøg om du har seneste udgave
  • Læs readme.txt i arkivet (hvis der er et)
  • Eventuelt lav en back-up af data + mysql
Detailjerede installations beskrivelser kan du se i XOOPS Operations Manual

Problemløsning


Jeg får en hvid side, kan ikke klikke brugernavn.......

  • Undersøg om din Xoops version > 2.5.0. Ældre versioner anvender ikke preloads, som er nødvendige.
  • Vælg debug i indstillingerne og se om der optræder fejl.
  • V.h.a FireFox, højreklik på skærmen, vælg inspect element, tryk på console i tabsene og se om der optræder nogle fejl (rød skrift)
  • Hvis ingen fejl, brug Firebug (Right click->inspect element->console) og se om der er jQuery fejl.
  • Hvis ingen jQuery fejl. Clean caches-> module update Smallworld, system modules.
  • Hvis stadig blank fejl, Gå til dit protector modul -> settings og skriv smallworld i formen "Modules out of Dos / Crawler checker".
