Aide : Waiting

Description

Ce module vous offre un bloc de contenu en attente extensible dans votre Xoops.

En installant ce module et uniquement en ajoutant les bons plug-ins, vous pouvez en tant que webmestre confirmer les attentes d'approbation de tous les modules d'un seul regard.

Installer / Désinstaller


Aucune mesure particulière n'est nécessaire, suivez le processus d'installation standard - extraire le dossier du module dans le répertoire ../modules. Installez le module à partir de l'Administration -> Module Système -> Modules.

Fonctionnement


Des instructions détaillées sur la configuration des droits d'accès des groupes d'utilisateurs sont disponibles dans le Manuel de fonctionnement de Xoops
