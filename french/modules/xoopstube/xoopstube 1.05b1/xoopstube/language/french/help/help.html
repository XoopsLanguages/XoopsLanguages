Aide : XoopsTube

Description


Avec le module XoopsTube, vous pouvez ajouter et classer les films provenant de sites comme YouTube, DailyMotion, LiveLeak, etc. sur votre site internet.

Installation / désinstallation


Aucune mesure spéciale n'est nécessaire. Suivez la procédure d'installation standard, envoyez le dossier « xoopstube » dans le répertoire .../modules. Installez le module par Admin-> Système-> Modules. Si vous avez besoin d'instructions détaillées sur la façon d'installer un module, veuillez consulter le Manuel des opérations de XOOPS.
