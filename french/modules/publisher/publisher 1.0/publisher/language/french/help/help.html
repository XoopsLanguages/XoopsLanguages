Aide : Publisher

Description


Le module Publisher est un système de gestion d'articles pour votre site Xoops.

Vous pouvez créer des articles dans diverses catégories

Mode d'emploi


Pour configurer ce module, vous devez :

a) Configurer vos préférences du module (voir l'onglet « Préférences » dans l'administration du module Publisher)

b) Créer des catégories afin de pouvoir classer vos articles (voir l'onglet « Catégories » dans l'administration du module Publisher)
