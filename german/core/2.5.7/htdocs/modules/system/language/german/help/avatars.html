Hilfe: Avatar

Beschreibung

Avatare sind kleine Grafiken oder Bilder Die man in verschiedenen Content Bereichen verwenden kann.Sie sind oft neben den Benutzernamen zu finden als Visuelle Identifizierung.Typische Anwendungen sind das Forum als Beispiel.

Der Avatar-Manager ermöglicht dem Administrator das Hinzufügen,bearbeiten oder löschen der System Avatare sowie Benutzer Avatare.

System Avatare

System Avatare sind Avatare,eingespielt durch den Administrator in den Bereich Avatar.

Benutzer Avatar

Benutzer Avatare werden durch die User selbst hochgeladen.Dieses muß separat im Adminbereich eingestellt und zugelassen sein.Hier kann auch zulässige Größe des Bildes bzw. Uploadgröße festgelegt werden

Bearbeiten und löschen der Avatare

Der Seitenadministrator kann alle System Avatare bearbeiten bzw.löschen.Hier ist zu beachten,das der User der so ein Avatar gewählt hat,seines dann verliert.
