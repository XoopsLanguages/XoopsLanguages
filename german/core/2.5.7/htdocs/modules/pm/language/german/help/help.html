Hilfe: Private Messaging

Beschreibung

Das PM Modul ermöglicht den Austausch Privater Nachrichten zwischen Benutzern

Installation/Deinstallation

Installation efolgt wie gehabt, dem standard installations Prozess folgen – Das Modul entpacken in das Verzeichnis ../modules. Installieren Sie das Modul via Admin -> System Module -> Module.

Ausführliche Anweisungen zur Installation von Modulen finden Sie im XOOPS Operations Manual

Konfiguration

Um das Modul einzurichten gehen Sie wie folgt vor:

  • Gehen Sie von der front page direkt zu Ihrem Posteingang, oder klicken Sie auf den Link "zum Module gehen" in der Administration von PM.
  • Die Voreinstellungen für das Module einrichten (unter ‘Einstellungen’) und optional einen PM Block einrichten (unter ‘Blöcke’).
  • Sie können mehrere Nachrichten auf einmal löschen indem Sie den "Säubern"-Tab im Menü klicken.
