Zusammenfassung

  • Diese Modul entwickelt ein Theme für den Administrationsbereich.

Install

  • Zum installieren eines Moduls benötigen Sie drei Schritte :
    • Hochladen des ThAdmin in dein Modulverzeichnis und das Modul Installieren.
    • In General Settings, select 'thamin' for CPanel GUI parameter

Hinzufügen eines neuen Theme

  • Kopiere das Voreingestellte Thema und Einfüge es ein und benenne es dann um
  • Verändere dieses neue erstellte Thema und aktuallisiere das Modul. Hierauf erschein dann das neue Theme mit den veräderten Moduleinstellungen.
