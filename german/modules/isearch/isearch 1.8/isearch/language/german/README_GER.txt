-- iSearch --  Analyse von User-Suchanfragen auf der Website

inl. dt. Sprachdateien


Version: 1.7

Autor: Hervé Thouzard ( http://xoops.instant-zero.com/ )



ZUSAMMENFASSUNG:

Das Modul ermöglicht es, die von Usern durchgeführten Suchanfragen auf der Seite zu protokollieren und auszuwerten. Neben einer umfassenden Statistikfunktion bietet das Modul auch eine Export- und Blacklist-Funktion. Admins können für die gesuchten Begriffe direkt auf die Ergebnisseiten zugreifen, um so ihr Webangebot zu optimieren.



VERWENDUNG:

Einfach installieren.
Es wird nur das Modul iSearch installiert, für das gegebenenfalls die Gruppenrechte angepasst und die Blöcke positioniert werden müssen. In der Navigation erscheint dann der neue Menüpunkt "iSearch".
