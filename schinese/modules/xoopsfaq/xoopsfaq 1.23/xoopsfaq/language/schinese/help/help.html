Help: XOOPS FAQ

Description


XOOPS FAQ is a XOOPS module to create FAQ (Frequently Asked Questions). You can create an unlimited number of questions and answers.

You can use a WYSIWYG editor to make your FAQ even more attractive

Install/uninstall


No special measures necessary, follow the standard installation process â€“ extract the /xoopsfaq folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Operating instructions


This module and its operations are very simple:

i) Configure your preferences (e.g. select the text editor)

ii) Add one or more Categories

iii) Add Questions and Answers

iv) Check that you have given your user groups the necessary module and block access rights to use this module. Group permissions are set through the Administration Menu -> System -> Groups.

Detailed instructions on configuring the access rights for user groups are available in the XOOPS Operations Manual
