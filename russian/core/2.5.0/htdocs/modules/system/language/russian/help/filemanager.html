Help: File Manager

Description

Using File Manager you have access to all the files within the XOOPS installation. You can do following:

- edit content of each file (if there is an editor available - you'll see the Edit icon)
- change Unix file permissions for each file
- delete a file
- create new directory
- create new file
- upload a file

Thanks to File Manager in XOOPS, you don't need FTP application for simple file operations/changes.
