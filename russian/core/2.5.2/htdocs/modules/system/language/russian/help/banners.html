Помощь: Банеры

Описание

Банеры объявлений отображаются в верхней части сайта. Они могут быть в виде картинок или флеш-анимации, например банер XOOPS, который устанавливается по умолчанию. XOOPS имеет встроенную систему управления банерами и создания учётных записей для клиентов-рекламодателей, которая находится в Модуль системы -> Банеры. Вы можете добавить несколько банеров на сайт, они будут чередоваться при обновлении страницы.

Каждый раз, когда отображается банер посетителям, это засчитывается как один "Показ", который является общей мерой для всех банеров. Клиенты обычно платят за определённое число показов - т.е., чтобы их банер отображался определённое число раз.

Примечание: Если вы делаете бизнес по показам банеров, то есть одна важная вещь, которую вы должны сделать: Выберите Модуль системы -> Конфигурация -> Основные настройки и заполните поле "Ваш IP адрес" (Если у Вас статический). Показы банеров с этого IP адреса не будут засчитываться. Если у Вас динамический IP адрес, то не заполняйте это поле.
