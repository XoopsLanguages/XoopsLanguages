XOOPS 安装说明

全新安装

前言:

XOOPS 是一个高度可扩展、面向对象的建站系统，基于PHP语言和MySQL数据库开发。

XOOPS 作为一个成熟的网站建设管理工具，可以方便的建设管理各类网站：内容管理与界面风格分离，便于方便灵活的设计切换不同的表现风格；成熟的功能模块化机制， 有各种丰富多样的模块可供选择定制，比如新闻发布、文章管理、论坛、相册、下载、友情链接、广告管理、黄页、在线购物、博客和维基；具有完善的权限管理机 制，可以针对不同用户群组设置不同权限。

XOOPS 可以作为理想的WEB二次开发平台，开发各种规模不同类型的动态网络社区，实现高度定制化的内容管理功能。

XOOPS 是一个自由软件，遵循GNU General Public License (GPL)协议，在此前提下，您可以按照自己的需要进行修改。

系统需求

  • WWW 服务器 (Apache, IIS, Roxen 等)
  • PHP 4.3 或更高版本（推荐使用 5.2 或更新版本）
  • MySQL 数据库服务器 3.23.XX 以上版本（推荐使用4.12或更高版本）

安装准备

  1. 正确设置WWW服务器，PHP和数据库服务器。
  2. 建立一个用于安装XOOPS的数据库。
  3. 配置用于登录数据库的帐号及密码。
  4. 设定 uploads/目录及 mainfile.php 属性为可写。
  5. 如果要安装GIJOE开发的protector模块，请把/extras/mainfile.dist.php.protector文件改名为mainfile.dist.php并拷贝替代htdocs/mainfile.dist.php
  6. 处于安全考虑建议把xoops_lib（公共类库）和xoops_data（XOOPS数据）放到document root路径之外，并且修改目录名。
  7. 设置xoops_data/目录可写； 生成并设置如下目录可写：xoops_data/configs/、xoops_data/caches/、xoops_data/caches/xoops_cache/、xoops_data/caches/smarty_cache/以及xoops_data/caches/smarty_compile/。
  8. 启用浏览器 cookie 及 JavaScript 功能。

安装到本地

如果为了开发或测试安装到本地环境中,首先要确认你运行的本地环境符合XOOPS的系统需求.然后从htdocs中复制文件到你的本地环境的根目录中,在浏览器中输入"http://yoursite.com".进入安装向导开始安装.

安装到服务器

如果需要安装到服务器中,首先在本地解压缩文件包或者使用在线工具解压缩.上传htdocs目录下全部文件至网站根目录.上传完成后,在浏览器中输入"http://yoursite.com".进入安装向导开始安装.

开始安装

只需按照安装向导提示安装即可.
