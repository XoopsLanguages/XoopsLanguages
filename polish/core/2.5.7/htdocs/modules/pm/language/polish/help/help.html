Pomoc: Prywatne Wiadomości

Opis

moduł PM jest do prywatnych wiadomości między użytkownikami

Zainstaluj / odinstaluj

Nie potrzeba żadnych specjalnych kroków. Należy tylko wykonać standardowy proces instalacji - wyodrębnić / PM folder do .. / modules. Zainstaluj moduł przez Admin -> Module System -> Moduły

Szczegółowe instrukcje dotyczące instalacji modułów są dostępne w XOOPS: Poradnik Obsługi

Instrukcja obsługi

  • Wejdź do skrzynki odbiorczej bezpośrednio na stronie głównej lub klikając na górnym linku "Przejdź do modułu" w PM Admin obszarze.
  • Skonfiguruj preferencje modułu (patrz: "Ustawienia") i ewentualnie blok PM, jeśli masz zamiar go używać (patrz "Bloki").
  • Możesz zrobić masowe usuwanie wiadomości za pomocą "Usuń Wiadomości" tab.
