Help: eXtGallery

Description

eXtGallery is a powerful web gallery module for XOOPS

Category/Albums


Modify or delete public category:
You can find a list of all categories/albums available on your web site. The number between the bracket at the end of the category name is the weight of the category.

Add a public category:
You can add a category only under a category without a photo. Only valid category are diplayed in the selection below.

Photos

Add batch photos:
Batch folder path : /opt/lampp/htdocs/local/test/tdm1/modules/extgallery/batch/
By pressing the SUBMIT button below, all photos in the batch folder will be uploaded to the album/albums selected below.

Rebuild Thumb:
Select the categorie where thumb will be rebuilt. This feature is usefull if you want to change thumb size or quality

Edit/Delete photo:
Select the category in which you wish to edit or delete photos.

Permissions

Select permission:
Please check all parts and update permission. If you want users can see your albums and photos you must select permission for all group

Public permissions mask:
Here you can set the default mask permissions that will be applied to a new category. Don't give over generous permissions here because all new categories will inherit them.

Access permission:
Allows you to restrict access to this category. A restricted user can't see a category unless they have permission set to do so.

Rating permission:
Allows you to restrict the ratings feature. A restricted user cannot rate in a category but can still see the rating for the image.

Public eCard permission:
Allows you to restrict the send eCard feature. All eCard's senders IP addresses are logged.

Public download permission:
Allows you to restrict the download of large image. If a user is allowed to download, they will also have access to large image downloads if the "Save large photo" is enabled in preferences.

Download original permissions:
Allows you to restrict the download of original images. If a user is allowed to download original they can access the original image (without border and watermark, and not resized) if the "Save original photo" is enabled in preferences.

Public upload permission:
Allows you to restrict the upload/submit of new images on public albums.

Public auto approve permission:
Allows you to prevent the automatic display of submitted public image until they are approved by an admin. Select category where public user uploads need not be approved. Public user's must be allowed to submit photos to enable this permission.

Display album permissions:
Allows you to not display category names but still allow image access. For example, you could use photos in these categories for your site content.
