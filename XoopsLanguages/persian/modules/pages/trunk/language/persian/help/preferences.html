Préférences du module

Comme la plupart des modules Xoops, celui ci n'échappe pas à la règle et vous propose un peu de paramétrage pour personnaliser différents éléments dont nous détaillons ici ceux qui méritent une explication complémentaire :

  • Répertoire d'upload des fichiers HTML : Indiquez ici le répertoire de destination des fichiers html qui seront uploadés. Le répertoire par défaut puisque déjà situé dans le répertoire uploads est censé déjà bénéficier d'autorisations de lecture-écriture(chmod >=775)
  • Répertoire de stockage des images uploadées : comme pour les fichiers html, précisez quel répertoire recevra les images uploadées. Le "I" majuscule du répertoire Image est lié à une contraite de FCK Editor
  • Les types MIME autorisés sont ceux qui figurent dans le fichier [root]/class/uploader.php
  • Afficher la section 'Import HTML' dans la création des pages : afin de ne pas surcharger l'interface de création des pages, vous pouvez désactiver cette option si elle ne vous est pas utile
  • Afficher un menu intégré ( liens sur les autres pages) : en activant cette option des liens vers les autres pages de ce module seront affichées àà la fin de la page. Cette option permet de favoriser votre référencement
  • Type d'affichage du menu intégré : pour le menu intégré décrit ci-dessus vous avez le choix d'afficher ces liens verticalement ou horizontalement
  • Autoriser les visiteurs anonymes à accéder à la page : faire un lien sur notre site ? c'est ici que vous gérez les autorisations sur les pages spéciales comme la page "informer un ami"
  • Type d'éditeur : des éditeurs wisiwyg ont été intégrés dans ce module, voir les explications
