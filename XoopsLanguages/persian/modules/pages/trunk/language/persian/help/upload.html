Upload

Cette page vous permet de réaliser les opérations suivantes :

  • upload d'images qui serviront d'en-tête au début de chaque page
  • upload de fichiers html réalisés avec votre éditeur favori pour intégration dans ce module
  • upload des images liées àà ces fichiers html

Dans les références du module, vous avez la possibilité de modifier les répertoires par défaut pour ces uploads, mais nous vous recommandons néanmoins de conserver ceux proposés par le module.

Si la valeur de serveur uploads est à ON c'est que le serveur sur lequel fonctionne votre site autorise cette opération. La valeur maximum d'un fichier en upload est également affichée sur cette page. Cette valeur est déterminée par un paramètre du fichier php.ini de votre serveur et si vous êtes sur un hébergement mutualisé, vous n'avez pas la capacité de modifier cette limite.
