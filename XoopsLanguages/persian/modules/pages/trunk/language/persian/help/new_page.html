Nouvelle page

Nous alkons décrire ici les différents éléments essentiels à la création d'une nouvelle page.

1. Ajouter une page

  • Image ou logo de la page : une liste déroulante vous propose d'afficher une image qui sera affichée avant le titre de votre article. Pour ajouter des images à cette liste, veuillez consulter la documentation. En choisissant l'image blank.png aucune image ne s'affichera
  • Poids de la page : donner une valeur à votre page, plus le chiffre sera faible plus cette page sera affiché en premier dans le menu intégré ou dans le bloc menu principal. Vous pouvez avoir une vision complète de l'ordre des pages créées avec le menu réordonner les pages
  • Titre du sous-menu : indiquez ici le nom du lien secondaire qui sera affiché dans le bloc menu principal. Il peut bien sur être différent du titre du document si vous le souhaitez
  • Titre de la page : c'est le nom qui sera affiché au début du document ainsi que dans la barre de titre du navigateur. Il sera également repris comme titre de la page pour le référencement

2. HTML et paramétrage du contenu

Cette partie concerne uniquement les informations à saisir pour l'utilisation du contenu d'un fichier html préalablement importé dans le module au moyen de la page

3. Contenu

C'est dans ce cadre que vous aurez à saisir le contenu de votre page. Les options de nettoyage qui vous sont affichées concernent principalement ceux qui font de l'import de fichier html ou du copier/coller d'un code source html issu d'un document word dont on connait le langage verbeux et inutile au sens strict html.

Si votre article est assez long, vous avez la possibilité d'utiliser la balise xoops [pagebreak]. Il suffit de la positionner dans votre texte à l'emplacement où vous souhaitez intégrer un saut de page. Ceci se traduira par un affichage des numéros de pages juste avant le début de chaque page et en base de celle-ci.

3. Menu

Vous disposez dans cet espace d'options permettant de gérer automatiquement la publication et le non affichage de votre article en fonction des dates saisies. C'est une option facultative, compléter ces options n'est pas obligatoire.

Si vous répondez oui à la question "définir comme page index du module", cela signifie que c'est cette page qui sera affichée si l'utilisateur clique sur le lien pages du bloc menu principal de xoops.

Une réponse positive à l'interrogation Ajouter un lien dans le sous-menu , aura pour conséquence d'ajouter un lien vers cette page dans le bloc menu principal. Ce sera un sous-lien du lien pages.

Ajouter cet article dans la page de lien ? (ou menu intégré ) : comme nous vous l'avons précisé dans la page preferences, il vous est possible d'ajouter un lien vers cette page dans le menu intégré.

4. Autorisations d'accès

Pour que la publication d'un article vous soit plus facile, les autorisations d'accès sur la page sont directement accessibles à partir de ce tableau.

5. Conseils

Pour réaliser des pages simples, fonctionnelles, agréables et qui soient référencées, préférer la rédaction de votre contenu au format html. Cela nécessite l'apprentissage de quelques balises simples (moins d'une dizaine) qui doivent vous satisfaire dans la majorité des cas

exemple

<h1>Titre de votre page</h1> (automatiquement pris en compte en remplissant le champ titre) <h2>sous titre 1</h2> <p>Commencez à écrire votre contenu dans une balise paragraphe que l'on ferme à la fin</p> <p>Pour écrire du texte avec de courtes descriptions utilisez les listes à puces : <ul> <li>premier item</li> <li>second item</li> </ul> </p> <p>Si vous avez besoin de faire un lien il suffit de l'écrire ainsi <a href="http://www.site.com/" title="titre du lien">lien</a></p> <p>Pour insérer une image, rien de plus facile avec la balise img <img src="http://www.site.com/images/monimage.jpg" width="100px" height="100px" alt="titre de mon image" /> </p> <p>Encore plus fort, réaliser un lien avec une image, il faut juste insérer le code de l'image à la place du nom du lien <a href="http://www.site.com/" title="titre du lien"><img src="http://www.site.com/images/monimage.jpg" width="100px" height="100px" alt="titre de mon image" /></a>
