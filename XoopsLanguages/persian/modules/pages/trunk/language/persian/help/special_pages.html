Pages spéciales

Ce module est une évolution du module wf-channel dans lequel ces pages existaient déjà, elles ont donc été conservées sans faire l'objet de modifications. Les autorisations d'accès pour ces deux pages se définissent dans les paramètres du module.

1. Page de lien

Cette page propose à vos visiteurs de faire d'insérer une bannière, un logo afin de faire un lien vers votre site.

Les images que vous pouvez sélectionner dans les différentes listes déroulantes doivent être présentes dans le répertoire uploads\pages\Image que vous pouvez alimenter soit avec un logiciel ftp, soit avec le menu upload du module.

A noter que dans cette page on indique à vos visiteurs qu'ils peuvent intégrer le contenu de ces pages dans un flux RSS ou à défaut avec un script javascript.

2. Informer un ami

Avec cette option, vous pouvez proposer à vos visiteurs un formulaire dans lequel ceux-ci peuvent laisser un message qui sera transmis par mail au destinataire indiqué.
