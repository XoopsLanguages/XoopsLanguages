توضیحات

این بخش در انجام چند عملیات اصلی شما را یاری میکند تا سلامت زوپس و سایت خود را حفظ نمایید:

- خالی کردن کش
- خالی کردن جدول session
- انجام تعمیرات از جدول پایگاه داده (تجزیه و تحلیل - بررسی - بهینه سازی و تعمیر)

شما همچنین میتوانید عملیات خالی کردن جداول را برای بک آپ شخصی انجام دهید.
