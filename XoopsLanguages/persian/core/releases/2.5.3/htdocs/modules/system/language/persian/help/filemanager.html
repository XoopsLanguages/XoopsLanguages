Help: File Manager

توضیح

در صورت استفاده از مدیریت فایل ها شما میتوانید به تمامی فایل هایی که توسط زوپس نصب شده است دسترسی داشته باشید:

- ویرایش مشخصات هر فایل (در صورتی که امکان ویرایش وجود داشته باشد شما گزینه ویرایش را مشاهده میکنید)
- تغییر دسترسی یونیکس فایل ها
- حذف فایل ها
- ایجاد مسیر جدید
- ایجاد فایل جدید
- بارگزاری فایل

در صورت استفاده از مدیریت فایل دیگر نیازی به استفاده از نرم اف تی پی نخواهید داشت
