Ayuda: Mantenimiento

Descripción

Mantenimiento ofrece varias funciones básicas para ayudarle manteniendo su instalación de XOOPS y mantenerlo saludable:

- limpiar el folder caché
- vaciar la tabla de sesiones
- realizar el mantenimiento de las tablas de base de datos (analizar, revisar, optimizar y reparar)

También puede realizar un "volcado" de la base de datos total o tablas individuales para los propósitos de copia de seguridad.
