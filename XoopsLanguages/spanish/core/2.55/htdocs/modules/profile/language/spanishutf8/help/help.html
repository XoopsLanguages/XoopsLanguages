Ayuda: Perfil

Descripción

El módulo perfil es para manejar campos personalizados del perfil de usuario.

Instalar/desinstalar

No se necesitan medidas especiales, siga el proceso de instalación estandard – extraiga la carpeta xoopspartners dentro del directorio ../modules. Instale el módulo a través de Admin -> Sistema de módulos -> Modulos.

Instrucciones detalladas para la instalación de módulos está disponible en el Manual de Operaciones de XOOPS

Instrucciones de Operación

Para utilizar este módulo necesita:

i) Configure sus preferencias para el módulo (vea ‘Preferencias’) y opcionalmente el bloque Amigos si desea usarlo (vea ‘Bloques’)

ii) Edite la categorías existentes o agregue nuevas.

iii) Edite los campos existentes o agregue nuevos. Aquí usted puede especificar cuales campos serán visibles en cual categoría, y si ellos serán visibles durante el registro de usuarios

iv) Defina el orden de los pasos de Registro

v) Y finalmente, puede poner permisos para campos individuales - cuales son editables, cuales son buscables
