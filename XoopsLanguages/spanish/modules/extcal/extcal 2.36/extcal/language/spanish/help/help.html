Ayude : extCal

Descripción

Módulo eXtCal es un calendario de eventos desarrollados exclusivamente para Xoops, por lo que está perfectamente integrado.

Utiliza la autorización y Xoops gestión de grupos, lo que permite una gran flexibilidad.

Instalar / desinstalar

No son necesarias medidas especiales. Siga el procedimiento de instalación estándar, envíe el archivo "eXtCal" en el directorio .../modules. Instale el módulo en Admin-> Sistema-> Módulos.

Instrucciones detalladas sobre cómo instalar un módulo sobre Operaciones Xoops Manual

Instrucciones

Para configurar este módulo, deberá:

  • i) Establezca sus preferencias para el módulo (consulte "Preferencias") y, posiblemente, el bloque de pareja si usted va a utilizar (vea "bloques")
  • ii) Compruebe que ha dado a los usuarios y grupos de bloques, los derechos de acceso para utilizar este módulo. Los permisos de grupo se definen en la Administración a módulos -> Sistema -> Grupos.

Información detallada sobre la configuración de los derechos de acceso para grupos de usuarios, disponibles en lasOperaciones Xoops Manual
