Ayuda: eXtGallery

Descripción

eXtGallery es un poderoso módulo de galería web para XOOPS

Categoría/Albumes


Moduficar o eliminar categoría pública:
Puede encontrar una lista de todas las categorías/albumes disponibles en su sitio web. El número entre el soporte al final del nombre de la categoría es el peso de la categoría.

Añadir una categoría pública:
Puede añadir una categoría solo bajo una categoría sin foto. Sólo se muestran categorías válidas en la lista de abajo.

Fotos

Añadir fotos por lotes:
Ruta de la carpeta batch (lotes): /opt/lampp/htdocs/local/test/tdm1/modules/extgallery/batch/
Al pulsar el botón ENVIAR a continuación, todas las fotos en la carpeta de los lotes será cargado en el álbum o álbumes seleccionados a continuación.

Reconstruir Miniaturas:
Seleccione la categoría donde se reconstruiran las miniaturas. Esta función es útil si desea cambiar el tamaño o calidad de las miniaturas

Editar/Eliminar foto:
Seleccione la categoría in la que desea editar o eliminar fotos.

Permisos

Seleccione el permiso:
Revise todas las partes y actualize los permisos. Si quiere que los usuarios puedan ver sus ábumes y fotos, debe selecionar permisos para todo el grupo

Máscara de permisos públicos:
Aquí puede poner las máscaras de permisos por defecto que se aplicarán a una nueva categoría. No de permisos muy generosos aquí porque lo heredarán todas las categorías nuevas.

Permisos de acceso:
Le permte restrinir elaccesoa esta categoría. Un usuario restringido no puede ver una categoría a menos que tengan permisos para hacerlo.

Permisos de valoración:
Le permite restringir la función de valoraciones. Un usuario restringido no puede valorar en una categoría pero todavía puede ver la valoración para la imágen.

Permiso público para enviar postáles:
Le permite restringir la función de envío de postales. Se registra la dirección IP de todos los que envíen postales.

Permiso público para descargar:
Le permite restringir la descarga de imágenes grandes. Si a un usuario se le permite descargar, también tendrán acceso para descargar imágenes grandes si se activa "Guardar foto grande"en preferencias.

Permisos para descargar original:
Le permite restringir la descarga de las imágenes originales. Si a un usuario se le permite descargar el original, ellos pueden accesar la la imágen original (sin borde ni marca de agua, y no redimencionada) si esta activo "Guardar foto original" en preferencias.

Permiso público para subir:
Le permite restringir la subida/envío de nuevas imágenes en albumes públicos.

Permiso público de auto aprobación:
Lepermite prevenir el mostrar automáticamente las imágenes púbicas enviadas hasta que se aprueben por un administrador. Seleccione la categoría donde las subidas públicas de los usuarios no necesitan aprobarse. El usuario público debe ser autorizado para enviar fotos para activar este permiso.

Mostrar permisos del álbum:
Le permite no mostrar los nombres de las categorías pero todavía permite el acceso a las imágenes. Ejemplo, For example, sed podría usar fotos en esta categoría para el contenido de su sitio.

Configuración del Album:
Seleccionando los efectos cambiará la forma de mostrar la foto cuando se hace clic en las miniaturas. Pueden seleccionarse efectos Ajax o Normal.

Normal : Sin efectos
Lightbox : usa el efecto Lightbox
Overlay : usa el efecto Overlay
Tooltip : usa el efecto Tooltip
Fancybox : usa el efecto Fancybox
PrettyPhoto : usa el efecto PrettyPhoto
