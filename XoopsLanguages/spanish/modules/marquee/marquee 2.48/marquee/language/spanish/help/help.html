Help: Marquee

Description


Marquee is a XOOPS module to create scrolling texts (marquees). You can create an unlimited number of marquees and use them in 4 blocks. Texts can scroll horizontally (from right to left or left to right) and vertically (from top to bottom and bottom to top). You can also set the loop, i.e. how many times should the marquee loop (from 1 to 100 times, and infinity). You can use a WYSIWYG editor to make your marquees even more attractive

Operating instructions


This module and its operations are very simple:

i) Configure your preferences (e.g. select the text editor)

ii) Add a Marquee

iii) Select a block you want this Marquee to be visible and set desired options

iv) Check that you have given your user groups the necessary module and block access rights to use this module. Group permissions are set through the Administration Menu -> System -> Groups.

Detailed instructions on configuring the access rights for user groups are available in the XOOPS Operations Manual
