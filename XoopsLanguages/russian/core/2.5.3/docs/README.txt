XOOPS 2.5.3 Final

The XOOPS Development Team is pleased to announce the release of XOOPS 2.5.3. .

This 2.5 series builds on the XOOPS 2.4.x series with objectives of usability improvements instead of architecture or API changes. This release is reverting fix for bug ID: 2672723 Changed the code in Install to omits the SQL command: "ALTER DATABASE ... CHARACTER SET ..."

If you have a fresh 2.5.2 install, you should reinstall it with 2.5.3.
