Установка XOOPS

Первоначальная установка

Предисловие:

XOOPS - это динамический, OO (Объектно-Ориентированный), основанный на открытом исходном коде скрипт портала, написанный на PHP. XOOPS поддерживает различные базы данных. XOOPS - это идеальный инструмент для создания небольших высоко-динамичных сообществ вэб-сайтов, внутренних порталов компании, корпоративных порталов, web-публикаторов и многого другого.

XOOPS выпускается на условиях GNU General Public License (GPL) и может свободно использоваться и модифицироваться. Он распространяется свободно до тех пор, пока вы соблюдаете условия распространения GPL.

Требования

  • Вэб-сервер (Apache, IIS, Roxen, etc)
  • PHP 5.2
  • MySQL 5.1

Перед установкой

  1. Правильно установите вэб-сервер, PHP и базу данных MySQL.
  2. Подготовьте базу данных для вашего XOOPS сайта, установите кодировку utf8.
  3. Создайте пользователя и предоставить ему доступ к базе данных.
  4. Дайте директории /uploads/ и файлу mainfile.php права на запись.
  5. Из соображений безопасности настоятельно рекомендуем переместить эти две папки /xoops_lib/ (для библиотек XOOPS) и /xoops_data/ (для данных XOOPS) за пределы корня сайта и сменить их имена.
  6. Дайте директории /xoops_data/ права на запись; Создайте (Если их нет) и дайте права на запись следующим папкам /xoops_data/configs/, /xoops_data/caches/, /xoops_data/caches/xoops_cache/, /xoops_data/caches/smarty_cache/ and /xoops_data/caches/smarty_compile/ .
  7. Включите в браузере куки и JavaScript.

Установка на локальной машине

Если Вы используете локальную машину для установки и тестирования, то убедите что Ваша машина удовлетворяет системным требованиям. Если всё нужное ПО установлено, то скопируйте содержимое каталога htdocs (Из архива установки XOOPS) в корневой каталог Вашего сайта. Для начала установки введите в браузере ссылку на Ваш сайт, например "http://yoursite.com". Это запустит процесс установки.

Установка на платформе хостинга

Распакуйте архив установки XOOPS на Вашей локальной машине, или прямо на сервере, если у вас есть SSH доступ. Далее переместите все файлы из директории htdocs в корвеной каталог Вашего сайта. Для начала установки введите в браузере ссылку на Ваш сайт, например "http://yoursite.com". Это запустит процесс установки.

Продолжение установки

Просто следуйте пошагово инструкии установке, и выполняйте все требования инсталятора.
