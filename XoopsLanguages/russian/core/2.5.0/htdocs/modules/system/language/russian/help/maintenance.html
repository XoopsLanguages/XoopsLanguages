Справка: Техническое обслуживание

Описание

Техническое обслуживание обеспечивает несколько основных функций для поддержания вашего сайта в рабочем состоянии:

- очистка папок кеша
- очистка таблицы сессий
- обслуживание таблиц базы данных (анализ, проверка, оптимизация и востановление)

Вы можете также создать дамп всей базы данных или отдельных её таблиц.
