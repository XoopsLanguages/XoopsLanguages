The place for XOOPS languages maintained by XOOPS Translation Team.
Each language is managed as a sub-project by lead authors.

Structure:

XoopsLanguages
-- dirnamea
----- core
----- modules
----- themes
-- dirnameb
----- core
----- modules
----- themes

Structure Example:
XoopsLanguages/
-- spanish/
----- core
------- branches
------- releases
--------- 2.3.1
------- trunk
--------- docs
--------- upgrade
--------- release_notes_spanish.txt
----- modules
--------- moduleA
----------- branches
----------- releases
----------- trunk
----- themes

Note: You can use "foo" directory example. Copy and rename it.
