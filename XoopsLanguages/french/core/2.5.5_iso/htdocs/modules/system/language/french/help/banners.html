Aide : Bannières

Description

Les bannières sont des publicités affichées généralement dans la partie supérieure d'un site. Elles peuvent être des images ou des animations flash d'une dimension de 468 x 60 pixels, comme par exemple celle de la Communauté Francophone qui fait partie de l'installation par défaut.

XOOPS possède une régie publicitaire de bannières qui permet de créer des comptes pour vos annonceurs. Elle est accessible depuis le module Système puis "Bannières".
Vous pouvez avoir plusieurs bannières sur un site sous XOOPS, l'affichage est alternatif à chaque chargement d'une nouvelle page.

À chaque fois qu'une bannière est cliquée par un visiteur, une comptabilisation du "clic / impression" est effectuée . Généralement les anonceurs rémunèrent les diffuseurs pour un certain nombre d'impressions afin que leur site soit visité ce même nombre de fois.

Remarque importante : Si vous vendez des espaces publicitaires au nombre d'impression, vous devez aller dans : module Système -> Préférences -> Paramètres Généraux et renseignez le champs "Votre adresse IP" par la vôtre. Ceci afin d'éviter de comptabiliser vos propres visites depuis une impression/clic sur les bannières. Ne pas effectuer cette manipulation peut être considéré par vos annonceurs comme malicieuse.
