Aide : Blocs

Description

Les blocs sont des éléments de contenu qui peuvent être totalement administrés depuis l'interface d'administration de XOOPS. Outre les blocs natifs des modules qui permettent le rendu d'un contenu défini par ces derniers, vous avez la possibilité de créer des blocs personnalisés pour y insérer tout type de contenu (texte, image, graphique, iframe, ....). Le contenu de ces blocs peut être styliser individuellements ou hériter de celui de votre site.

Les blocs sont un atout précieux dans le rendu accessible du contenu de votre site. Chaque bloc peut être positionné suivant votre convenance à différents emplacements de votre site en fonction des modules affichés. Il est à noter tout de même, que certains blocs de modules sont conçus pour être affichés sur la colonne la plus large de votre page.
Nous pouvons considérer que la structure d'un thème pour XOOPS, possèdent trois secteurs d'affichage du contenu, une colonne à gauche, une colonne centrale dans laquelle s'affiche également le contenu à proprement parler d'un module, et une colonne à droite. L'ordre de ces colonnes dans les thèmes peut varier suivant la structure particulière définie par le webdesigner.

Il est tout à fait possible de déplacer les blocs sur un autre emplacement par simple glisser/déposer.
Lorsque vous installez un module, celui-ci peut ajouter ses propres blocs. La gestion de ceux-ci se réalise comme tout autre bloc de contenu.

Chaque bloc peut être configuré pour être visible sur certains modules et d'autres non.
Le "poids" permet de définir sa hiérarchie de positionnement par rapport aux autres blocs dans le même emplacement. Plus le poids est élevé, plus sa position sera "basse".

Si vous désirez affichez un même bloc dans des secteurs/colonnes différentes suivant les modules, vous devez créer un "clone" de ce bloc. Ceci se réalise tout simplement en cliquant sur l'icône de clonage.
