Bienvenue dans le Centre d'Aide de XOOPS

Comment puis-je utiliser XOOPS ?

XOOPS est un Système Dynamique de Gestion de Contenu, Extensible (OO Orienté Objet) et facile à utiliser, écrit en PHP et utilisant également MySQL pour les données. Pour l'utiliser vous avez besoin d'installer des modules afin de lui apporter des fonctionnalités complémentaires : articles, forums, galerie de photographie, boutique e-commerce, gestionnaires d'évènements, etc .... Ces modules peuvent être téléchargés sur notre Référentiel.

Comment utiliser le Centre d'Aide ?

Le Centre d'Aide vous apporte les informations basiques afin de paramétrer les préférences de votre site, et d'utiliser les fonctions de base. Chaque fois que vous souhaitez en savoir un peu plus sur la section où vous vous trouvez, il vous suffit de cliquer sur l'icône d'aide dans le coin droit.

Comment insérer du contenu ?

Vous avez besoin d'un module complémentaire pour cela. Il peut être le module "News, "Content", "Publisher", ...

Comment avoir plus d'aide ?

Vous pouvez obtenir de l'aide complémentaite en visitant notre forum de support officiel ou celui de votre communauté linguistique.
