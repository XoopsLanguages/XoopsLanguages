Aide : Messagerie Utilisateurs

Description

XOOPS intègre un système interne de messagerie privé qui vous permet d'envoyer un message à vos utilisateurs inscrit dans votre base de données. Cet envoi peut se faire sur la totalité des inscrits, pour certains groupes d'utilisateurs uniquement, ou sur une sélection en utilisant des filtres de dates par exemple.
Cela peut prendre l'aspect d'un message privé (via le système interne de messagerie de Xoops) ou d'un message par email.

Vous pouvez également utiliser des variables prédéfinies qui vous permettront de personaliser vos messages.
