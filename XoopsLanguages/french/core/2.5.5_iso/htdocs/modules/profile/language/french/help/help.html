Aide : Profile

Description

Le module Profil permet de gérer des champs personnalisées pour les informations sur les utilisateurs.

Installion / Désinstallation

Aucune mesure spéciale n'est nécessaire, suivre le processus de base d'installation -> installation et désinstallation du module via l'interface d'administration de ces derniers Administration -> Module System -> Modules.

Un tutoriel détaillé d'installation des modules dans Xoops est disponible à partir de ce lien Manuel d'Utilisation de XOOPS

Instruction d'usage

Pour utiliser ce module, vous avez besoin de :

  • Configurer les préférences du module et optionnellement l'affichage d'un bloc si vous désirez l'utiliser,
  • Editer les catégories existantes ou en ajouter une nouvelle,
  • Editer des champs de saisie existants ou en ajouter des nouveaux. Vousp ouvez préciser quels champs seront visibles en fonction des catégories, et s'ils doivent être visibles lors du processus d'inscription d'un utilisateur,
  • Définir l'ordre des différentes étapes du processus d'inscription / enregistrement,
  • Finalement, paramétrer individuellement les permissions pour certains champs - certains peuvent être éditables ou être dans la fonction de recherche / tri par exemple
