Aide : Classements des utilisateurs

Description

Le système de classement (que nous pourrions également nommer : rang, position, ..) permet de donner par exemple une "hiérarchie" ou une "ancienneté" ou une "fonction" à vos utilisateurs, par défaut il est basé dans XOOPS sur le nombre de contribution sur le contenu du site.
Il est important de noter que les classements sont juste des "titres", ils ne donnent aucun privilège spécial ni de permission particulière. Le classement est totalement indépendant des groupes de permissions.
Si vous donnez le classement Administrateur à un utilisateur, mais que celui-ci n'est pas rattacher au groupe Administrateur, il n'aura donc pas les permissions d'administrer le site.

Comme indiqué plus haut, le classement par défaut est basé sur le nombre de contributions. Le classement de l'utilisateur sera actualisé automatiquement dès qu'il aura atteint le nombre de contributions requis pour le classement supérieur. Les rangs par défaut de XOOPS sont :

  • Débutant
  • Aspirant
  • Régulier
  • Semi Pro
  • Accro

Il intègre également deux classements spéciaux :

  • Modérateur
  • Administrateur

Un classement spécial diffère dans le sens où il peut être attribué sans tenir compte des critères requis pour le classement de base (dans la cas de XOOPS - sur les contributions). Une utilisation typique de classement spécial, est de donner par exemple une reconnaissance à des donateurs ou des utilisateurs ayant des fonctions particulières.

Vous pouvez éditer les classements par défaut, en cliquant sur le lien "éditer", vous changer le nom ou modifier les critères requis, ou le définir comme classement "spécial" ou non).
Vous pouvez personnaliser les classements en leur accordant des images représentatives. Vous pouvez également supprimer les rangs inutiles (lien "supprimer").
De même, vous pouvez ajouter un nombre illimité de classement suivant vos besoins. Pour cela, il vous suffira de renseigner les champs du formulaire de création.
