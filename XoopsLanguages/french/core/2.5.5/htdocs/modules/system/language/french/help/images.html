Aide : Gestionnaire d'images

Description

Le gestionnaire d'image est un outil vous permettant de télécharger, d'organiser et de récupérer des images pour les nécessités de votre site. Pour l'utiliser, vous devez préalablement le configurer, cela implique au minimum la création d'une catégorie afin de pouvoir stocker vos images, et de définir les préférences de celle-ci, comme par exemple : les dimensions maximales, la taille maximale du fichier ainsi que les permissions d'accès.
Chaque catégorie est configurable de manière indépendante les unes des autres.

D'un simple clic de souris, les images peuvent être catégorisées et téléchargées directement depuis le gestionnaire d'images par le biais de votre navigateur internet, ce qui permet l'insertion aisée d'images dans les articles ou autres contenus.
Il y a également un système extensif de permissions afin de contrôler les dimensions de l'image, la taille du fichier et les permissions d'accès des groupes d'utilisateurs aux différentes catégories.
