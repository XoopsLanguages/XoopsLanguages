Aide : Emoticônes

Description

Les émoticônes (ou "binette" - "smiley") sont des petites représentations graphiques que les utilisateurs, pour exprimer une émotion, peuvent utiliser en les insérant n'importe où dans le texte d'un article, d'un commentaire ou dans toute autre contribution.
XOOPS est livré avec une collection de 17 émoticônes. Elles sont accessibles dans la majorité des formulaires de rédaction des contenus.
En cliquant sur une émoticône, celle-ci est insérée dans le texte à l'emplacement désiré.
Les émoticônes peuvent également être insérés en saisissant une combinaison de caractères, par exemple en saisissant :-D dans une contribution cela insèrera cette émoticône :

La liste complète des émoticônes est disponible sur votre système XOOPS et les combinaisons de caractères sont accessibles depuis le gestionnaire des émoticônes dans cette partie de l'administration de XOOPS.
Vous pouvez ajouter, modiffier ou supprimer des émoticônes, et contrôler celles qui doivent être affichéee sur les formulaires (par défaut, 9 des 17 sont accessibles).

Si vous ne voulez pas utiliser d'émoticônes sur votre site, vous pouvez désactiver cette fonction en cochant la case "Afficher dans le formulaire". Dans ce cas, elles n'apparaîtront plus dans les formulaires mais resteront tout de même disponibles par la combinaison de caractères. Une autre méthode est de supprimer simplement l'émoticônes.

Pour les désactiver totalement cette fonctionnalité (formulaire et combinaisons de caractères), vous devez vous rendre dans les Préférences du Panneau de Contrôle, puis dans les Paramètres du module système.
