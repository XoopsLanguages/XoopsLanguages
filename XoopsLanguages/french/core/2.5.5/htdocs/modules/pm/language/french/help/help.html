Aide : Private Messaging

Description

Le module PM offre la possibilité aux utilisateurs de s'adresser entre eux des messages privés

Installion / Désinstallation

Aucune mesure spéciale n'est nécessaire, suivre le processus de base d'installation -> installation et désinstallation du module via l'interface d'administration de ces derniers Administration -> Module System -> Modules.

Un tutoriel détaillé d'installation des modules dans Xoops est disponible à partir de ce lien Manuel d'Utilisation de XOOPS

Instruction d'usage

Pour utiliser ce module :

  • Vous entrez dans votre messagerie directement de la partie "publique" du site ou en cliquant sur le lien "Allez au mocule" dans l'interface d'administration du module,
  • Configurer les préférences du module et optionnellement l'affichage d'un bloc si vous désirez l'utiliser,
  • Vous pouvez effacer en nombre les message en vous rendant sur la page "Purge des messages"
