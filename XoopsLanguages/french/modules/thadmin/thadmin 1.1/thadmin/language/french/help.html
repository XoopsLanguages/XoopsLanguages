Présentation

  • ThAdmin permet de gérer l'apparence de votre interface d'administration

Installation

  • Pour installer le module, respecter les étapes suivantes :
    • Placer le dossier /thadmin dans le répertoire des modules de votre site et installez-le depuis le panel d'administration des modules
    • A la rubrique Paramètres généraux, selectionner 'thamin' comme Interface du Panneau de configuration

Ajouter un nouveau thème

  • Copiez / collez puis renommez le Thème Default
  • Personnalisez votre thème et mettez à jour le module. Votre nouvelle interface d'administration est disponible depuis les options du module ThAdmin
