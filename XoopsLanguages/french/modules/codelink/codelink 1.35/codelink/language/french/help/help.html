Help: codelink

Description

Modulo per la visualizzazione dei links, dei bottoni e dei banners del sito, prelievo del codice per lo scambio con altri siti.

Install/uninstall

No special measures necessary, follow the standard installation process - extract the /codelink folder into the ../modules directory. Install the module through Admin -> System Module -> Modules. If you need detailed instructions on how to install a module, please see the XOOPS Operations Manual.
