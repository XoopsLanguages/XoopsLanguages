Aide : Google Maps

Description

Google Maps crée une carte du monde Google, où les utilisateurs peuvent ajouter des marqueurs pour les emplacements.

Installation / Désinstallation

Aucune mesure spéciale n'est nécessaire, suivez le processus d'installation standard - extrayez le dossier du module dans le répertoire « modules ».
Installer le module via Admin -> System Module -> Modules.

Des instructions détaillées sur l'installation des modules sont disponibles dans le Manuel des opérations de Xoops

Mode opératoire

Pour mettre en place ce module, vous devez :

  • Configurer vos préférences pour le module (voir « Préférences »), et insérer votre clé API Google.
    Récupérez cette clé en suivant ce tutoriel : https://developers.google.com/maps/documentation/javascript/tutorial?hl=fr-FR#api_key
    Clé pour les applications du navigateur (avec referers) Clé API

  • Ajoutez vos catégories.
  • Ajoutez vos points.
  • Ajoutez vos lignes polygonales

Notes :

  • La première catégorie sera l'affichage par défaut de la carte. Cela peut être modifié en réorganisant les catégories.
  • Les images et les liens peuvent être ajoutés dans les marqueurs en utilisant la saisie de texte xoops comme d'habitude.
  • Les lignes polygonales relient les marqueurs ensemble.
