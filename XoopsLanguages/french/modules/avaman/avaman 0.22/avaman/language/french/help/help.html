Help: Avaman

Description


Le module Avaman gère les Avatars et les smileys pour l'utilisateur

Installer/désinstaller


Aucune mesure particulière n'est nécessaire, suivez le processus d'installation standard, envoyez le dossier « avaman » dans le répertoire /modules. Installez le module par Admin-> système-> Modules. Si vous avez besoin d'instructions détaillées sur la façon d'installer un module, veuillez consulter le Manuel des opérations de XOOPS.
