Aide : Birthday

Description

Birthday est un module pour suivre les anniversaires de vos membres.

Il peut également être utilisé pour se souvenir des soldats morts au combat, en indiquant la date de leur passage. Vous pouvez utiliser un éditeur WYSIWYG pour le rendre encore plus attrayant

Installation / Désinstallation

Aucune mesure particulière n'est nécessaire. Suivre le processus d'installation standard : envoyer le dossier /birthday dans le répertoire .../modules. Installer le module par Admin-> système-> Modules.

Des instructions détaillées sur l'installation de modules sont disponibles dans le Manuel des opérations de XOOPS

Instructions d'exploitation

Ce module et ses opérations sont très simples.

Des instructions détaillées sur la configuration des droits d'accès pour les groupes d'utilisateurs sont disponibles dans le Manuel des opérations de XOOPS
