aChat 0.2

Bienvenue dans aChat, un module de Chat, Tagboard avec AJAX!

Editez les 'Préférences' pour définir vos paramètres personnels. 

Pour cloner le module, allez dans la partie 'Utilitaires' et choisissez un nom du module clone.
Le clonage du module aChat permet par exemple d'utiliser un module par salon, thème de discussion.
