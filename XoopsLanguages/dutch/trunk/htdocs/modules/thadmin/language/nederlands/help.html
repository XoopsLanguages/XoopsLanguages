Beschrijving

  • Deze module zorgt ervoor dat je een uiterlijk voor de XOOPS administratie kan kiezen

Instalatie

  • Voor het instaleren moet je drie stappen volgen :
    • Upload de ThAdmin map in de module map van xoops.
    • Verander in de Algemene Instellingen de administratie GUI in "ThAdmin"

Een nieuw thema toevoegen

  • Maak een kopie van het standaard thema, en hernoem het naar de naam van je nieuwe thema
  • Bewerk dit thema,upload het in de goede map, update ThAdmin en nou kan je je thema in de lijst selecteren
