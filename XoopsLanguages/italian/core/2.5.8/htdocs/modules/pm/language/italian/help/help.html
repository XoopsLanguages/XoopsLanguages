Help: Messaggi Privati

Descrizione

Il modulo PM è per la messaggistica privata tra utenti

Installa/Disinstalla

Nessuna misura particolare, seguire il processo di installazione standard ed "estrarre la cartella /pm" nella directory ../modules. Installare il modulo attraverso Admin -> System Module -> Modules.

Istruzioni dettagliate su come installare i moduli sono disponibili nello XOOPS Operations Manual

Istruzioni per l'uso

Per impostare questo modulo è necessario:

  • Si entra in Posta in arrivo direttamente dalla prima pagina del modulo, o cliccando sul link in alto "Vai al modulo" nella zona di Admin del modulo PM.
  • Configurare le preferenze per il modulo (vedi "Preferenze") e, facoltativamente, i blocchi del modulo pm, se si intende utilizzarli (vedere "Blocchi").
  • Si possono cancellare in massa i messaggi utilizzando la scheda "Pulizia messaggi".
