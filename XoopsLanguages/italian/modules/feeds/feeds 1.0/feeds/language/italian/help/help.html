Help: feeds

Descrizione

Questo modulo raccoglie attraverso plugin rss, i dati dei singoli moduli.
Se utilizzi un url esterno inserendo il link nel campo di testo Rss Url, non hai bisogno di utilizzare i plugin del modulo
Un url esterno potrebbe essere anche un file del modulo, ad esempio: http://www.tuosito.org/modules/TDMDownloads/rss.php

Installa/Disinstalla

Misure speciali necessarie, non seguire il processo di installazione standard - estrarre la cartella /feeds nella directory ../modules. Installare il modulo attraverso Admin -> System Module -> Modules. Se avete bisogno di dettagliate istruzioni su come installare un modulo, si prega di consultare il XOOPS Operations Manual.
