Help: Avaman

Description


Avaman Module manages Avatars and Smilies for the user

Install/uninstall


No special measures necessary, follow the standard installation process, extract the "avaman" folder into the ../modules directory. Install the module through Admin -> System Module -> Modules. If you need detailed instructions on how to install a module, please see the XOOPS Operations Manual.
