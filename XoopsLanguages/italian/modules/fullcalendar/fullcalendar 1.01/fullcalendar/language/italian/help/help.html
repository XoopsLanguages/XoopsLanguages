Help: fullcalendar

Description

fullcalendar is a module for write your events, based on jquery and jquery ui

Install/uninstall

No special measures necessary, follow the standard installation process - extract the /fullcalendar folder into the ../modules directory. Install the module through Admin -> System Module -> Modules. If you need detailed instructions on how to install a module, please see the XOOPS Operations Manual.
