Help: testadsl

Description

Con il nostro tool potrete misurare subito e gratuitamente la velocità del vostro collegamento internet e ADSL.

Install/uninstall

No special measures necessary, follow the standard installation process - extract the /testadsl folder into the ../modules directory. Install the module through Admin -> System Module -> Modules. If you need detailed instructions on how to install a module, please see the XOOPS Operations Manual.
