Aiuto: IP Log

Descriptione

IP Log è un modulo per registrare gli indirizzi IP solo degli utenti, oppure gli utenti ed ospiti anonimi.

Installa/disinstalla

Misure speciali necessarie, non seguire il processo di installazione standard - estrarre /htdocs nella cartella XOOPS ROOT Path. Installare il modulo attraverso Admin -> System Module -> Modules. Se avete bisogno di dettagliate istruzioni su come installare un modulo, si prega di consultare il XOOPS Operations Manual.

Assicurarsi di installare la chiave API di IPInfoDB.com nelle preferenze altrimenti il modulo non funziona, Questo avviene correttamente andando su www.ipinfodb.com registrarsi, e dopo aver confermato il tuo indirizzo e-mail, ti verrà fornita la chiave API per incollarla nel suo campo nelle preferenze.
