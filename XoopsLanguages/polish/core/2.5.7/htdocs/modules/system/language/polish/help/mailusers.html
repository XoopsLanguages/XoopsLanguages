Help: Email Users

Description

XOOPS posiada wbudowany system komunikacji, który umozliwia wysylanie wiadomosci do calej bazy zarejestrowanych uzytkowników lub ich podgrup, lub przy uzyciu filtrów daty. Moze to byc w formie prywatnej wiadomosci (tj. przez XOOPS wewnetrzny system wiadomosci) lub przez elektroniczna email. XOOPS ma rózne opcje dla filtracji podgrupy uzytkowników z calego czlonkostwa.

Mozna równiez uzyc predefiniowanych znaczników, które pozwola Ci dostosowac Twoja wiadomosc.
