MultiMenu
Version: 1.7
Developer: Wolfactory
Developed At: http://www.wolfpackclan.com/wolfactory 
Introduction
 This module is inspired by iMenu from luinithil, which allow to create custom menus. 
 
 Purpose
 MultiMenu is used to create custom menus with links defined by the administrator. Link to anywhere on your own site or other sites in orderly menus 
 
 Key Features
 
- 4 menu blocks, customisable and configurable independently
- Link to anywhere on the web
