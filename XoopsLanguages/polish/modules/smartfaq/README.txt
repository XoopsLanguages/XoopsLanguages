Introduction
 Module for displaying questions and answers for frequently asked questions (FAQs)
 
 Purpose
 When you have basic information on the form question => answer, it may be beneficial to use a FAQ module for organising this information
 
 Key Features
 
- FAQ categories
- User submissions
- ?Open questions? for users to answer
- Question and answer moderation
