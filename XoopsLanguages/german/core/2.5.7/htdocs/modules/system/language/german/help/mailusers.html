Hilfe: E-Mail User

Beschreibung

In XOOPS ist ein Nachrichten System integriert,mit dem Sie eine Nachricht an Ihre gesamten registrierten Benutzer und Gruppen schicken können,die entsprechend gefiltert werden können.Dies kann in Form einer Privaten Nachricht oder über eine E-Mail erfolgen.Eine Reihe von Optionen gibt es für das Herausfiltern der Gruppen,der Benutzer in der Mitgliedschaft.

Sie können auch vordefinierte Tags verwenden,um Ihre Nachricht personalisieren.
