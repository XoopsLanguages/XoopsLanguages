Hjælp: Profile

Beskrivelse

Profilmodulet er til at håndtere tilpassede brugerprofil felter.

Installer/afinstaller

Ganske alm installerings fremgangsmåde – Udpak /profile mappen til ../modules directory. Installer modulet gemmem admin -> System Module -> Modules.

Detailjerede instrukser om installering af moduler kan findes på XOOPS Operations Manual

Operations instrukser

For at opsætte dette modul skal du:

  • Konfigurere dine prefencer for modulet (se ‘Preferences’) og eventuelt Profile bloken hvis du vil bruge denne (se ‘Blocks’).
  • Ændre eksisterende kategorier eller lav nye.
  • Ret i eksisterende felter eller lav nye. Her kan du specificere hvilke felter der skal være synlige i hvilken kategori, og om de skal være synlige gennem registreringen.
  • Definer ordenen af registreringens trin.
  • Og sidst kan du sætte tilladelser for individuelle felter - hvilken der skal kunne rettes og hvilke der er søgbare.

Anti-Spam muligheder

For at minimere spam registreringer skal du gøre følgende:

  • Gå til Protector modulet it Admin, gåtil Preferences, og til sidst ved denne option: "Stop Forum Spam" Checker POST data og om spammeren er registreret på www.stopforumspam.com database.
    Set denne til "Ban the IP (no limit)"

  • i /class/captcha/config.php, check at mode er sat til "text":

    return $config = array(
    'disabled' => false, // Slår CAPTCHA fra
    'mode' => 'text', // default mode, du kan vælge 'text', 'image', 'recaptcha' (fordrer api nøgle)
    'name' => 'xoopscaptcha', // captcha navn
    'skipmember' => true, // Springer CAPTCHA over checker for medlemmer
    'maxattempts' => 10, // Max forsøg i hver session
    );

  • I Profile modulet, gå til Admin og i Basis trinet på:
    /modules/profile/admin/step.php?id=1
    sæt "Gem efter stop" til "Nej"


  • I Profil Præferencer, sæt "Brug Captcha efter andet registreringstrin" til "Ja" (dette er default)
