Beskrivelse

Vedligeholdelse giver flere grundlæggende funktioner til at hjælpe dig med at opretholde din XOOPS installation og holde det sundt:

  • Du kan rense cache mappe.
  • Du kan tømme session tabellen.
  • udføre vedligeholdelse af database tabeller (analysere, kontrollere, optimere og reparation).

Du kan også udføre en "Dump" af hele databasen eller enkelte tabeller til backup formål.
