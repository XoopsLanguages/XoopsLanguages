Summary

  • This module provides a theme for your administration area.

Install

  • For install the module, you have just three step :
    • Upload ThAdmin into the module directory and install the module.
    • In General Settings, select 'thamin' for CPanel GUI parameter

Add a new theme

  • Copy/paste and rename the default theme
  • Modify your theme and update the module. The theme appear inside the module settings
