Ayuda: Mensaje Privado

Descripción

El módulo PM es para enviar mensajes entre los usuarios

Instalar/desinstalar

No son necesarias medidas especiales, siga el proceso estandard de instalación – extraiga la carpeta /pm en el directorio ../modules. Instale el módulo a través de Admin -> System Module -> Modules.

Están disponibles las instrucciones detalladas en la instalación de módulos en el Manual de operaciones de XOOPS

Instrucciones de operación

Para usar este módulo necesita:

i) Ingrese a su Bandeja de Entrada directamente desde la página principal, o haciendo clic en el enlace de arriba "Ir al módulo" en el área de administración de PM

ii) Configure sus preferencias para el módulo (vea ‘Preferencias’) y opcionalmente el bloque PM si quiere usarlo (vea ‘Bloques’)

iii) Puede hacer eliminación masiva de sus mensajes usando la pestaña "Purgar mensajes"
