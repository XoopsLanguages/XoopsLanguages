Help: Mensajería Privada

DESCRIPCIÓN

El módulo MP es para mensajes privados entre usuarios

INSTALAR/DESINSTALAR

No se requieren medidas especiales, siga el proceso de instalación estándar: extraiga la carpeta /pm en el directorio ../modules. Instale el módulo a través de Admin -> Módulo de Sistema -> Módulos.

Las instrucciones detalladas sobre la instalación de módulos están disponibles en el Manual de Operaciones de XOOPS Manual

INSTRUCCIONES DE OPERACIÓN

Para configurar este módulo, debe:

  • Ingresa su Bandeja de Entrada directamente desde la página principal, o haciendo clic en el enlace superior "Ir al módulo" en el área de Administración de MP.
  • Configure sus preferencias para el módulo (consulte "Preferencias") y, opcionalmente, el bloque MP si tiene la intención de usarlo (consulte "Bloques").
  • Puede borrar en masa sus mensajes usando la pestaña "Separar mensajes".
