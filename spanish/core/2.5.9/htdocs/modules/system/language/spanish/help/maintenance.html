Ayuda: Mantenimiento

Descripción

El mantenimiento proporciona varias funciones básicas para ayudarlo a mantener su instalación de XOOPS y mantenerla en buen estado:

- Limpiar carpeta de caché
- Vaciar la tabla de la sesión
- realizar el mantenimiento de las tablas de la base de datos (analizar, verificar, optimizar y reparar)

También puede realizar un "volcado" de toda la base de datos o tablas individuales para realizar copias de seguridad.
