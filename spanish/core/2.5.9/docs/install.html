Como Instalar XOOPS

Primera Instalación (Nueva)

Prefacio:

XOOPS es de código libre,un Sistema de Publicación Web Orientado a Objetos escrito en PHP. Es una herramienta ideal para desarrollo de sitios web comunitarios dinámicos pequeños y grandes, portales intraempresa (intranet), portales corporativos, weblogs y mucho más.

XOOPS se lanza bajo los términos de la Licencia pública general de GNU (GPL) versión 2 o superior, y es de uso y modificación gratuitos. Es libre de redistribuir siempre que respete los términos de distribución de la GPL.

Requisitos

  • Servidor Web WWW(Apache, NGINX, IIS, etc)
  • PHP 5.3.7 o mayor, 5.6+ es recomendado
  • MySQL 5.1 o mayor, 5.6+ es recomendado

Antes de instalar

  1. Configure el servidor WWW, PHP y el servidor de base de datos correctamente.
    2.         
  2. Prepare una base de datos para su sitio XOOPS.
    3.         
  3. Prepare la cuenta de usuario y otorgue al usuario el acceso a la base de datos.
    4.         
  4. Hacer que los directorios y los archivos se puedan escribir: %s
    5.         
  5. Por consideraciones de seguridad, se recomienda encarecidamente mover los dos directorios siguientes fuera del directorio raíz y cambie los nombres de las carpetas: %s
    6.         
  6. Crear (si no está ya presente) y hacer que los directorios se puedan escribir: %s
    7.         
  7. Active las cookies y JavaScript de su navegador.

Instalación Local

Si está ejecutando un entorno local para desarrollo o prueba, asegúrese de tener los anteriores requisitos cumplidos. Una vez hecho esto, copie el contenido del directorio htdocs (desde el archivo de distribución de XOOPS) a la ruta del directório raíz de su entorno web. Una vez que los archivos se copien allí, puede iniciar la instalación apuntando su navegador a "http://localhost/susitio". Esto iniciará el proceso de instalación.

Instalación en una plataforma alojada (Web)

Si está ejecutando en un entorno alojado, descomprima los archivos XOOPS localmente o en el servidor si tiene directorio htdocs a su directorio web raíz (su proveedor generalmente proporciona direcciones a esta ubicación). Una vez que los archivos se copien allí, puede iniciar la instalación apuntando su navegador a "http://susitio.com". Esto iniciará el proceso de instalación.

Continuando con la instalación

Simplemente siga las instrucciones paso a paso en el instalador.
