Help: Perfil

DESCRIPCIÓN

El módulo Perfil es para administrar campos personalizados del perfil de usuario.

INSTALAR/DESINSTALAR

No se requieren medidas especiales, siga el proceso de instalación estándar: extraiga la carpeta /profile en el directorio ../modules.Instala el módulo a través de Admin -> Módulo de Sistema -> Módulos.

Las instrucciones detalladas sobre la instalación de módulos están disponibles en Manual de Operaciones XOOPS

INSTRUCCIONES DE OPERACIÓN

Para configurar este módulo, debes:

  • Configure sus preferencias para el módulo (consulte 'Preferencias') y, opcionalmente, el bloque de Perfil si tiene la intención de usarlo (consulte 'Bloques').
  • Edite categorías existentes o agregue nuevas.
  • Edite campos existentes o agregue nuevos. Aquí puede especificar qué campos serán visibles en qué categoría y si serán visibles durante el registro del usuario.
  • Defina el orden de los pasos de Registro.
  • Y, por último, puede establecer permisos para campos individuales, cuáles son editables y cuáles se pueden buscar.

Medidas AntiSpam

Para minimizar los registros de spam, haga lo siguiente:

  • vaya al módulo Protector en Admin, vaya a Preferencias, y luego en la parte inferior, en esta opción: "Stop Forum Spam" Comprueba los datos POST contra los spammers registrados en la base de datos de www.stopforumspam.com.
    Establézcalo en "Prohibir el IP (sin límite)"

  • en /class/captcha/config.php, asegúrese de que el modo esté configurado como "texto":

    return $config = array(
    'disabled' => false, // Deshabilitar CAPTCHA
    'mode' => 'text', // modo predeterminado, puede elegir 'text', 'image', 'recaptcha'(requiere api key)
    'name' => 'xoopscaptcha', // nombre de captcha
    'skipmember' => true, // Omita el cheque CAPTCHA para miembros
    'maxattempts' => 10, // Máximos intentos para cada sesión
    );

  • En el módulo Perfil, vaya a Admin y en el paso Básico ubicado en:
    /modules/profile/admin/step.php?id=1
    selecciona el "Guardar después de parar" a "No"


  • En Preferencias de perfil, configure la opción "Utilizar Captcha después del segundo paso de registro" en "Sí" (es el valor predeterminado)
