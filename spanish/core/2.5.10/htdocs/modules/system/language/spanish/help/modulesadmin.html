Ayuda: Módulos

Descripción

Los módulos son las construcciones de software que, aprovechando la infraestructura que proporciona XOOPS, permiten que el administrador ofrezca a los usuarios del sitio contenido dinámico o estático generado mediante una interfaz web que con mayor frecuencia no es muy fácil de usar.

Parte de la potencia de XOOPS radica en la forma en que se gestionan los módulos. El administrador puede instalar, desinstalar, activar o desactivar fácilmente cualquier módulo específico (excepto el módulo del sistema, por razones obvias). También es posible cambiar el nombre de los módulos y determinar el orden en el que aparecen al usuario final en el menú del sitio (si desea que aparezcan), porque también puede proporcionar funcionalidad oculta, definiendo módulos activos que no tienen presencia visible para el usuario ocasional. Por ejemplo, puede definir un módulo como no visible en el menú desactivando el icono de verificación en la columna de Menú. Por lo tanto, para el módulo Noticias, puede enviar mensajes privados al grupo seleccionado de sus usuarios y decirles que accedan a ese directorio escribiendo la dirección en el campo URL, algo así como http://www.yoursite.com/modules/news.

Para cambiar el orden de los módulos (que se reflejarán en el Menú), solo tiene que arrastrar y colocar los módulos en la ubicación deseada.

La instalación del módulo necesita una cierta planificación, tanto para los módulos que se mostrarán como para el diseño de las páginas. Es una buena idea revisar otros sitios XOOPS y ver qué módulos se usan para mostrar varios tipos de contenido. Esto se hace fácilmente abriendo cualquier sitio XOOPS y pasando el cursor sobre el menú enlaza los paneles laterales izquierdo y derecho. Al mismo tiempo, observe las URL que se muestran en la barra de estado de su navegador. Debería ver direcciones como esta: http: //el_sitio.com/modules/news / .... Todos los módulos están almacenados en el directorio /modules/, por lo tanto, el siguiente sector de la dirección (en este ejemplo, "news") muestra el módulo que mostrará los contenidos de ese enlace de menú.

Si abre la página para el enlace, se mostrará un ejemplo de cómo se usa el módulo y el tipo de contenido que muestra. Revisar otros sitios como este también le darán una buena idea de las capacidades de diseño.
