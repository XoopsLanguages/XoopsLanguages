Bienvenido al: Centro de Ayuda de XOOPS

¿Cómo uso XOOPS?

XOOPS es una plataforma de aplicaciones web escrita en PHP para la base de datos MySQL. Su orientación a objetos lo convierte en una herramienta ideal para desarrollar sitios web de comunidades pequeñas o grandes, portales intranet de compañías y corporativos, weblogs y mucho más. Para usar XOOPS de manera efectiva, necesita instale módulos que brinden la funcionalidad que está buscando. Hay muchos módulos diferentes en el Repositorio de Módulos, desde calendarios, galerías de fotos, noticias, tablones de anuncios, hasta actualizaciones del clima.

¿Cómo uso el Centro de ayuda?

El Centro de ayuda es para proporcionarle información básica relacionada con las características individuales de la Administración. Siempre que desee obtener más información sobre la página en particular en la que se encuentra, simplemente haga clic en el ícono de ayuda en la esquina derecha

¿Cómo puedo crear contenido en XOOPS?

Necesitará un módulo para eso. Esto podría ser un módulo de noticias o un módulo de contenido.
