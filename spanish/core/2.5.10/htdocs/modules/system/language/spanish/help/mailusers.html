Ayuda: Correo a Usuarios

Descripción

XOOPS tiene un sistema de mensajería integrado que le permite enviar un mensaje a toda su base de usuarios o subgrupos de usuarios registrados, o usar filtros de fecha. Esto puede ser en forma de un mensaje privado (es decir, a través del sistema de mensajería incorporado de XOOPS) o mediante un correo. Una gama de opciones está disponible para filtrar subgrupos de usuarios de su membresía.

También puede usar etiquetas predefinidas que le permitirán personalizar su mensaje.
