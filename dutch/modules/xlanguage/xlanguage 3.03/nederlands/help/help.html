Help: xLanguage

Omschrijving

Een uitgebreid Multi-taal beheer plugin voor inhoud- en karaktercodering binnen XOOPS websites.

Multitaal beheer zorgt voor het weergeven van de inhoud van verschillende talen, zoals Engels, Frans en Chinees.

Karaktercodering beheer zorgt voor de inhoud van andere coderingsets voor één taal , zoals GB2312 (Vereenvoudigd Chinees) en BIG5 (Traditioneel Chinees) voor Chinees.

Wat kan met xlanguage wel?

1 Het weergeven van inhoud van een opgegeven taal op basis van de gebruiker zijn keuze

2 het omzetten van inhoud van de ene karakterset naar de andere .

Wat kan met xlanguage NIET?

1 xlanguage heeft niet de mogelijkheid van het vertalen van de inhoud van de ene taal naar een andere. De inhoud van de verschillende talen dient door de beheerder zelf aangemaakt te worden

2 xlanguage werkt niet zonder het toevoegen van een regel aan XOOPS/include/common.php (Zie handleiding hieronder)

3 xlanguage heeft niet de mogelijkheid van het omzetten van de inhoud van de ene karakterset naar een ander als "iconv", "mb_string" of "xconv" niet beschikbaar is.

Functies

1 auto-detectie van taal voor bezoeker bij eerste bezoek

2 Bewaren van gebruiker zijn taal instellingen

3 Schakelen tussen verschillende taal/karaktersets

4 Ondersteuning van M-S-M mode voor karaktercodering

Opmerkinge:

M-S-M: Multiple encoding input, Single encoding storage, Multiple encoding output. M-S-M staat een site toe waar verschillende gebruikers andere taal/karaktersets gebruiken. Bijvoorbeeld, een site heeft xlanguage geimplementeerd en stelt gebruikers in staat om content toe te voegen ofwel met GB2312, met BIG5 of UTF-8 codering en deze content op te slaan in de database met specifieke codering, zoals GB2312, en vervolgens weer te geven met GB2312, BIG5 of UTF-8 codering.

Installeren/de-installeren

Geen speciale maatregelen zijn nodig, volg de standaard installatieprocedure – pak de /xlanguage map uit in de ../modules map. Installeer de module via Administratie -> Modules -> Systeembeheer -> Modules.

Gedetailleerde instructies voor het installeren van modules zijn beschikbaar in de XOOPS Gebruikershandleiding

Gebruiksaanwijzing

1 installeer "xlanguage" als elk andere module

2 select basic langauges (from an available language list) and add extended languages (upon a selected basic language) from module admin page
for instance, to make language switch between: English, Simplified Chinese (gb2312), Traditional Chinese (big5) and UTF-8 Chinese:
base 1: name: english; description(optional): English; charset: iso-8859-1; code: en (or anyother like "xen", not a true language code, just the tag for indicating English content)
base 2: name: schinese; description(optional): Simplified Chinese; charset: gb2312; code: zh (or anyother like "sc", not a true language code, just the tag for indicating Chinese content)
extended lang of schinese 1: name: tchinese; description(optional): Traditional Chinese; charset: big5 code: zh-TW (the true language code of Traditional Chinese) base: schinese
extended lang of schinese 2: name: utf8; description(optional): Simplified Chinese UTF-8; charset: utf-8 code: zh-CN (the true language code of Simplified Chinese) base: schinese

3 maa het blok "Taalselectie" zichtbaar

4 add multilingual content with according tags sepcified for each base language (in step 3) to your modules, templates or themes[Skip this step if you do not use multi-language content display but only use charset encoding]:
wrap content of each language with respective tag specified in step 3:
[langcode1]Content of the language1[/langcode1] [langcode2]Content of the language2[/langcode2] [langcode3]Content of the language3[/langcode3] ...
if two or more languages have same content, you do not need add them one by one but use delimiter "|":
[langcode1|langcode2]Content shared by language1&2[/langcode1|langcode2] [langcode3]Content of the language3[/langcode3] ...
a true example (suppose the lang_codes specified in step 4 are: English-en; French-fr; SimplifiedChiense-sc):
[en]My XOOPS[/en][fr]Moi XOOPS[/fr][sc]我的XOOPS[/sc]
OR:
[english|french]This is my content in English and French[/english|french][schinese]这是我在中国的内容[/schinese]

5 automatic conversion of content from one charset(extended language) to another [Actually on action needed in this step]

6 __if__ you would like to insert hardcoded scripts for language switch in your theme or any template besides the language selection box:
1) modify /modules/xlanguage/api.php "$xlanguage_theme_enable = true;"
2) config options "$options = array("images", " ", 5); // display mode, delimitor, number per line";
3) insert <{$smarty.const.XLANGUAGE_SWITCH_CODE}> into your theme or template files anywhere you prefer it present
