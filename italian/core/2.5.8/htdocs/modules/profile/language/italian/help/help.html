Help: Profile

Descrizione

Il modulo Profile è per la gestione di campi del profilo utente personalizzate.

Installa/Disinstalla

Nessuna misura particolare, seguire il processo di installazione standard ed "estrarre la cartella /profile nella directory ../modules. Installare il modulo attraverso Admin -> System Module -> Modules.

Istruzioni dettagliate su come installare i moduli sono disponibili nello XOOPS Operations Manual

Istruzioni per l'uso

Per impostare questo modulo è necessario:

  • Configurare le preferenze per il modulo (vedi "Preferenze") e, facoltativamente, i blocchi del modulo profile, se si intende utilizzarli (vedere "Blocchi").
  • Modifica categorie esistenti o aggiungerne di nuove.
  • Modifica campi esistenti o aggiungerne di nuovi. Qui è possibile specificare quali campi saranno visibili in quale categoria, e se sarà visibile durante la registrazione utente.
  • Definire l'ordine delle fasi di registrazione.
  • E, infine, è possibile impostare le autorizzazioni per i singoli campi - quali sono non modificabili, quali sono ricercabili.
