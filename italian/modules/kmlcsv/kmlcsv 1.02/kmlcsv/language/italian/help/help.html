Help: kmlcsv

Description

The KmlCsv modules is a conversion type of Kml to Csv and Csv to Kml

Install/uninstall

No special measures necessary, follow the standard installation process - extract the /kmlcsv folder into the ../modules directory. Install the module through Admin -> System Module -> Modules. If you need detailed instructions on how to install a module, please see the XOOPS Operations Manual.
